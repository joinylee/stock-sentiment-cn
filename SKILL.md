# A股市场情绪分析 (Stock Sentiment CN)

## Overview

分析中国 A 股市场情绪的综合工具，集成雪球、东方财富、同花顺等数据源，帮助投资者了解市场整体情绪状态。

## Quick Start

```bash
python3 scripts/analyze.py --report
```

## Commands

### 生成完整报告
```bash
python3 scripts/analyze.py --report
```

### 查询雪球热门
```bash
python3 scripts/analyze.py --hot
```

### 市场情绪指标
```bash
python3 scripts/analyze.py --sentiment
```

### 板块情绪分析
```bash
python3 scripts/analyze.py --sectors
```

### 社交媒体情绪
```bash
python3 scripts/analyze.py --social
```

## Features

### 📊 整体市场情绪
- **恐惧贪婪指数** (0-100)
  - 0-30: 极度恐惧（可能底部区域）
  - 30-50: 偏恐惧
  - 50-70: 偏乐观
  - 70-100: 极度贪婪（风险区域）

- **上涨/下跌比例**
- **主力/散户资金流向**
- **换手率**
- **波动率**

### 🔥 雪球热门讨论
- 热门股票排行榜
- 讨论热度评分
- 个股涨跌表现

### 🏭 板块情绪
- 行业资金流向
- 板块涨跌对比
- 主力加仓/减仓行业

### 💬 社交媒体情绪
- 雪球舆论分析
- 微博多空对比
- 热门话题追踪
- 关键词云分析

### 💰 资金流向
- 主力资金动向
- 散户资金动向
- 北向资金（外资）
- 融资融券数据

## 情绪指标解读

### 恐惧贪婪指数 (Fear & Greed Index)

| 指数范围 | 情绪 | 含义 | 策略建议 |
|---------|------|------|---------|
| 0-20 | 极度恐惧 | 市场极度悲观 | 分批建仓优质股 |
| 20-40 | 恐惧 | 市场情绪低迷 | 小仓位试探 |
| 40-60 | 中性 | 多空平衡 | 观望或持有 |
| 60-80 | 贪婪 | 市场情绪过热 | 逢高减仓 |
| 80-100 | 极度贪婪 | 市场狂热 | 清仓或做空 |

### 资金流向信号

| 信号 | 主力资金 | 散户资金 | 市场含义 |
|-----|---------|---------|---------|
| 吸筹 | 净流入 | 净流出 | 机构建仓 |
| 派发 | 净流出 | 净流入 | 机构出货 |
| 震荡 | 交替 | 交替 | 方向不明 |

## 投资建议

### 市场底部信号
- 恐惧贪婪指数 < 30
- 主力资金开始净流入
- 成交量极度萎缩
- 散户情绪极度悲观

### 市场顶部信号
- 恐惧贪婪指数 > 75
- 主力资金持续净流出
- 成交量暴增
- 新股民大量入场

## 数据来源

- 雪球 (xueqiu.com) - 散户情绪
- 东方财富 (eastmoney.com) - 资金流向
- 同花顺 (10jqka.com.cn) - 行情数据
- 新浪财经 - 新闻舆论
